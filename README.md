# 账单拆分插入工具

这是一个无需安装 Microsoft Excel 的 Windows 桌面工具。它按销售出库表 F 列的“客户全称 + 月份”拆分数据，并把每组数据插入指定的账单表头表尾模板中。

## 使用方法

1. 运行 `账单拆分插入工具.exe`。
2. 选择待拆分的销售出库文件（`.xlsx` 或 `.xls`）。
3. 选择表头表尾来源：软件填写，或从文件获取。
4. 选择输出目录。
5. 点击“开始执行”。执行期间可以点击“暂停”，再次点击“继续”。

表头表尾支持两种来源：

- **软件填写**：不显示模板路径，使用内置模板样式；填写公司抬头、账单年月（如 `2026-01`）和底部日期（如 `2026-01-31`）。底部日期会同步更新附注中的截止日期。
- 账单年月失去焦点后，底部日期会自动调整为该月最后一天；如果输入了 `2026-06-31` 等超出当月天数的日期，也会自动修正为 `2026-06-30`，并正确处理闰年 2 月。
- 底部日期按 `2026年6月30日` 文本写入，避免某些 WPS 或预览器显示成 Excel 日期序列号 `46203`。
- 软件会在当前 Windows 用户的 AppData 中保存上次使用的文件路径、输出目录、表头来源模式、公司抬头和日期设置，并在下次启动时自动恢复。
- **从文件获取**：显示表头表尾格式文件路径，输出内容完全取自所选模板，不应用软件填写项。

界面中的两个“下载模板”按钮可以随时导出程序内置的示例文件。输出文件统一为 `.xlsx`，工作表名称为“销售出库明细”。

仓库中的 `sample_data/202601销售出库.xlsx` 是脱敏示例，只包含虚构客户和订单；真实销售数据不会纳入 Git。

## 数据规则

- 源文件第一行为标题，数据从第二行开始。软件会根据业务表头自动识别列位置。
- 同时支持原始扩展结构（拆分键 F，数据 D/E/H–Q）和紧凑结构（拆分键 C，数据 A/B/E–N，例如 `202606销售出库.xlsx`）。
- 拆分键为空时，工具会使用识别到的“客户全称 + 月份标题”生成拆分键。
- 识别到的客户全称会写入账单 B6，日期、单号、客户简称、产品、数量、单价、金额和备注按表头映射输出。
- 模板第 1–7 行作为表头，第 9–15 行作为表尾；“软件填写”模式会覆盖公司抬头、账单年月、底部日期和附注截止日期，“从文件获取”模式保持模板原文。
- 表头中的 C、D 占位列会按旧宏逻辑删除，后续标题自动左移。
- 数据区原有公式会随列位置自动平移；H、J 合计保留 `SUM` 公式。
- 公式单元格同时写入计算缓存值，因此无需先用 Excel/WPS 重算也能看到合计结果。

## 源码运行与打包

```powershell
python -m pip install -r requirements.txt
python app.py
python build.py
```

打包结果位于 `dist\账单拆分插入工具.exe`。程序图标、示例销售表和表头表尾模板会被嵌入 EXE。
