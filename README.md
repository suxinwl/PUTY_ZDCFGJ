# 账单拆分插入工具

这是一个无需安装 Microsoft Excel 的 Windows 桌面工具。它按销售出库表 F 列的“客户全称 + 月份”拆分数据，并把每组数据插入指定的账单表头表尾模板中。

## 使用方法

1. 运行 `账单拆分插入工具.exe`。
2. 选择待拆分的销售出库文件（`.xlsx` 或 `.xls`）。
3. 选择表头表尾格式文件（`.xlsx` 或 `.xls`）。
4. 选择输出目录。
5. 点击“开始执行”。执行期间可以点击“暂停”，再次点击“继续”。

界面中的两个“下载模板”按钮可以随时导出程序内置的示例文件。输出文件统一为 `.xlsx`，工作表名称为“销售出库明细”。

仓库中的 `sample_data/202601销售出库.xlsx` 是脱敏示例，只包含虚构客户和订单；真实销售数据不会纳入 Git。

## 数据规则

- 源文件第一行为标题，数据从第二行开始，至少需要 A–Q 列。
- F 列作为拆分键；F 为空时，工具尝试使用 `G列客户全称 + F1月份`。
- 输出字段依次取源文件的 D、E、H、I、J、K、L、M、N、O、P、Q 列。
- 源文件 G 列客户全称写入账单 B6。
- 模板第 1–7 行作为表头，第 9–15 行作为表尾；模板中的月份、日期、制表人和说明文字不会被软件修改。
- 表头中的 C、D 占位列会按旧宏逻辑删除，后续标题自动左移。
- 数据区原有公式会随列位置自动平移；H、J 合计保留 `SUM` 公式。
- 公式单元格同时写入计算缓存值，因此无需先用 Excel/WPS 重算也能看到合计结果。

## 源码运行与打包

```powershell
python -m pip install -r requirements.txt
python app.py
python build.py
```

打包结果位于 `dist\账单拆分插入工具.exe`。程序图标、示例销售表和表头表尾模板会被嵌入 EXE。
